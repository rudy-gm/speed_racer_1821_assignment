Design Discussion

Our Speed Racer fan site applies several core design principles to create a cohesive, accessible, and engaging experience:

  1. Typography & Font Choice: We selected the Orbitron font to evoke a futuristic, high-speed racing vibe. Headings use clear hierarchy (display-4 for main, h3 for sections) to guide the user’s eye.
  2. Color & Contrast: A dark gradient background combined with bright text ensures high contrast for readability and a dramatic “night-race” atmosphere. Buttons and links use accent colors sparingly to draw attention without overwhelming.
  3. Layout & Responsiveness: Bootstrap’s grid system enables each page to reflow smoothly across mobile, tablet, and desktop. We used utility classes (margins, paddings, floats) for consistent spacing and image positioning.
  4. Imagery & Motion: Floating images on alternating sides create a sense of movement, mirroring Speed Racer’s dynamic races. Consistent image sizing (using w-75) maintains visual balance.
  5. Consistency & Branding: The fixed navbar and footer appear on every page, reinforcing navigation and brand identity. Iconography (“⚡Mach 5⚡”) remains the same across pages for coherence.
  6. Accessibility: Semantic HTML (proper headings, lists, sections) and meaningful alt text make content screen-reader friendly. We tested color contrast and keyboard navigation to meet WCAG 2.1 AA standards.
  7. Performance: Images are optimized and sized appropriately. We leveraged Bootstrap’s CDN for CSS and JS to improve load times and caching.
